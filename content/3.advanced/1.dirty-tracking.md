### _fields et _changes

[Contenu à remplir]

```ts
const user = await User.query().findByKey(1)

// État initial
console.log(user._fields) // { firstname: 'John', ... }
console.log(user._changes) // {}

// Modification
user.firstname = 'Jane'

console.log(user._changes) // { firstname: 'Jane' }
```
