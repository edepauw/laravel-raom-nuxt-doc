### Utiliser le fake mode dans les tests

[Contenu à remplir]

```ts
describe('User', () => {
  beforeEach(() => {
    User.fake().clear()
  })
  
  it('should find user by key', async () => {
    User.fake().seed([{ id: 1, firstname: 'John' }])
    const user = await User.fake().findByKey(1)
    expect(user.firstname).toBe('John')
  })
})
```
