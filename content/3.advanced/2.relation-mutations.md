## Attach

[Contenu à remplir]

### Attacher un élément

[Contenu à remplir]

```ts
user.posts.attach(post)
await user.save()
```

### Attacher plusieurs éléments

[Contenu à remplir]

```ts
user.posts.attach([post1, post2, post3])
await user.save()
```

### Attacher avec données pivot

[Contenu à remplir]

```ts
user.roles.attach(role, { expires_at: '2024-12-31' })
await user.save()
```

## Detach

[Contenu à remplir]

### Détacher un élément

[Contenu à remplir]

```ts
user.posts.detach(post)
await user.save()
```

### Détacher par ID

[Contenu à remplir]

```ts
user.posts.detach(3)
await user.save()
```

### Détacher tous les éléments

[Contenu à remplir]

```ts
user.posts.detach()
await user.save()
```

## Sync

[Contenu à remplir]

### Synchroniser avec un ensemble d'IDs

[Contenu à remplir]

```ts
user.roles.sync([1, 2, 3])
await user.save()
```

### Sync sans détacher

[Contenu à remplir]

```ts
user.roles.sync([4, 5], { withoutDetaching: true })
await user.save()
```

## Toggle

[Contenu à remplir]

### Basculer la présence d'un élément


### Toggle multiple

[Contenu à remplir]

