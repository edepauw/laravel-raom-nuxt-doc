# Installation

Get started with Laravel REST API Object Mapper in minutes.

## Prérequis

Before installing, make sure you have:

- **Node.js** 18.x or higher
- **Nuxt 3** application
- A Laravel API using [`lomkit/laravel-rest-api`](https://github.com/lomkit/laravel-rest-api)

## Installation via npx

The fastest way to install:

```bash [Terminal]
npx nuxt module add laravel-raom-nuxt
```

This command will:

1. Install the package
2. Add the module to your `nuxt.config.ts`
3. Create default configuration

## Installation manuelle

If you prefer manual installation:

```bash [Terminal]
npm install laravel-raom-nuxt
```

Then add the module to your `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ["laravel-raom-nuxt"],
});
```

## Configuration

### Configuration du module

The module currently has no configuration options. Simply register it in `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ["laravel-raom-nuxt"],
});
```

### Configuration de l'API backend

Make sure your Laravel backend has [`lomkit/laravel-rest-api`](https://github.com/lomkit/laravel-rest-api) installed and configured.

Install the package in your Laravel project:

```bash [Terminal]
composer require lomkit/laravel-rest-api
```

Publish the configuration:

```bash [Terminal]
php artisan vendor:publish --provider="Lomkit\Rest\RestServiceProvider"
```

Create REST resources for your models:

```php [app/Rest/UserResource.php]
<?php

namespace App\Rest;

use Lomkit\Rest\Http\Resource;

class UserResource extends Resource
{
    public static string $model = \App\Models\User::class;
}
```

## Vérification de l'installation

Create a simple model to verify everything works:

```ts [models/User.ts]
import {
  Model,
  Resource,
  Field,
  Key,
} from "laravel-raom-nuxt/runtime";

@Resource("users")
class User extends Model {
  @Key() @Field() id!: number;
  @Field() name!: string;
  @Field() email!: string;
}

export default User;
```

Test the connection:

```ts [pages/index.vue]
<script setup lang="ts">
import User from '~/models/User'

const user = await User.query().findByKey(1)
console.log(user.name)
</script>
```

If you see the user's name in the console, you're all set! 🎉
