## Inclusion de relations

[Contenu à remplir]

### Include simple

[Contenu à remplir]

```ts
const user = await User.query()
  .include('posts')
  .findByKey(1)
```

### Include avec contraintes

[Contenu à remplir]

```ts
const user = await User.query()
  .include('posts', (q) => q.orderBy('created_at', 'desc'))
  .findByKey(1)
```

### Include imbriqués

[Contenu à remplir]

```ts
const user = await User.query()
  .include('posts', (q) => q.include('comments'))
  .findByKey(1)
```
