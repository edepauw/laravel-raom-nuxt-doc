## HasMany

[Contenu à remplir]

### Définir une relation HasMany

[Contenu à remplir]

```ts
@Resource('users')
class User extends Model {
  posts = HasMany(() => Post, 'posts')
}
```

### Accéder aux éléments

[Contenu à remplir]

```ts
const user = await User.query().include('posts').findByKey(1)
const posts = user.posts.get()
```

## BelongsTo

[Contenu à remplir]

### Définir une relation BelongsTo

[Contenu à remplir]

```ts
@Resource('posts')
class Post extends Model {
  author = BelongsTo(() => User, 'author')
}
```

### Accéder à l'élément parent

[Contenu à remplir]

```ts
const post = await Post.query().include('author').findByKey(1)
const author = post.author.get()
```
