### Installation

[Contenu à remplir]

```bash
# Cloner le repository
git clone https://github.com/your-username/laravel-rest-api-object-mapper.git

# Installer les dépendances
npm install

# Préparer les stubs du module
npm run dev:prepare
```

### Développement

[Contenu à remplir]

```bash
# Playground de développement
npm run dev

# Build du playground
npm run dev:build
```

### Tests

[Contenu à remplir]

```bash
# Lancer tous les tests
npm run test

# Mode watch
npm run test:watch

# Lint
npm run lint
```
